# Brand Kit — Quick Reference

## Logos (`brand-kit/logos/`)

All SVGs, fully transparent, infinitely scalable.

| File | When to Use |
|------|-------------|
| `clock-lobster-icon.svg` | Default — red on dark bg |
| `clock-lobster-icon-white.svg` | White on colored/photo bg |
| `clock-lobster-icon-dark.svg` | Dark on white/light bg |
| `clock-lobster-horizontal.svg` | Website header, email signature |
| `clock-lobster-stacked.svg` | Hero section, print |
| `clock-lobster-social-avatar.svg` | Social profile pictures |
| `clock-lobster-favicon.svg` | Browser favicon (32x32 or 64x64) |

## Social Templates (`brand-kit/social/`)

Export to PNG before uploading.

| File | Size | Platform |
|------|------|----------|
| `twitter-header.svg` | 1500x500 | X header |
| `facebook-cover.svg` | 820x312 | Facebook cover |
| `instagram-post.svg` | 1080x1080 | Instagram post |

## Colors

- **Lobster Red:** `#e11d48` — logo, CTAs, accents
- **Deep Charcoal:** `#121214` — backgrounds
- **Slate:** `#1e293b` — card surfaces
- **Moonlight:** `#f8fafc` — primary text
- **Mist:** `#94a3b8` — secondary text

## Fonts

- **Outfit 700** — headings, logo text
- **Inter 400/500** — body copy
